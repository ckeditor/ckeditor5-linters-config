Added a new `ckeditor5-rules/ck-content-variable-name` rule for CSS files. Inside a `.ck-content` selector every `var(...)` is validated individually and must use the `--ck-content-*` prefix. Exemptions are configurable via the `ignoredVariableSubstrings` option.
