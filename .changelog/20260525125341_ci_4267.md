---
type: Major breaking change
scope:
  - eslint-config-ckeditor5
---

The preset now natively lints CSS files via [`@eslint/css`](https://www.npmjs.com/package/@eslint/css). All `**/*.css` files are linted using the `css` language, and the new `ckeditor5-rules/ck-content-variable-name` rule is enabled by default for `**/theme/**/*.css`.
