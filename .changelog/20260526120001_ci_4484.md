Added a new `ckeditor5-rules/no-disallowed-color-formats` rule for CSS files. The rule rejects hex colors, `rgb()`/`rgba()` calls, and named CSS colors in declaration values, restoring the bans previously provided by stylelint's `declaration-property-value-disallowed-list` and `color-named: "never"`.
